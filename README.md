## Descripción
Proyecto para la Asociacion de Nais y Pais del colegio Montevixia.

## Instalación
1. Clonar el proyecto

 https://github.com/AidaSousa/montevixia-new.git
```
2. Instalar dependencias
```
 composer install
```
3. Abrir el servidor local
```
 php artisan serve
```
